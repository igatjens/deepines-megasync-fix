Español

Deepines MEGAsync fix es un parche que repara un bug de MEGAsync en Deepin 20, el cual no permite que MEGAsync se abra y ejecute correctamente.

Este parche consiste en cambiar los parámetros de ejecución en /usr/share/applications/megasync.desktop. El parche se verifica en cada inicio del sistema para mantener MEGAsync funcionando después de cada actualización.

Puede desinstalar Deepines MEGAsync fix desde el lanzador de aplicaciones de Deepin, con clic derecho, desinstalar.

English

Deepines MEGAsync fix is a patch that repairs a MEGAsync bug in Deepin 20, which does not allow MEGAsync to open and run correctly.

This patch consists of changing the execution parameters in /usr/share/applications/megasync.desktop. The patch is checked at every system boot to keep MEGAsync working after every update.

You can uninstall Deepines MEGAsync fix from Deepin's application launcher, with right click, uninstall.
